# Game Art Direction

## Style
Isometric game art, Landstalker Sega Genesis style, painterly hand-painted art style, gouache brushstrokes.

## Projection
- True isometric, 2:1 width-to-height ratio on the ground plane
- Camera angle: ~35.26 degrees from horizontal, looking down from upper-right
- Light source: upper-left, shadows cast toward lower-right

## Palette
Warm greens, mossy earth tones, dappled golden light. Muted but saturated — not cartoonishly bright, not desaturated.

## Rendering
Hand-painted feel, NO photorealism. Visible brush strokes, soft blending, rich color variation. Watercolor meets gouache.

## Output
- Square image, 1024x1024
- Background color: solid magenta (#FF00FF) — used as a chroma key for transparency in post-processing
- Game asset with clean edges, suitable for compositing
- The asset itself must NOT contain magenta (#FF00FF)
