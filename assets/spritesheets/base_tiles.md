# Base Tiles

4 columns x 8 rows, 32 tile slots. Each cell is 256x128 containing a 2:1 isometric diamond.

## Row 0 — Grass:
- (0,0) **Grass 1** — lush green forest floor, short grass with subtle variation
- (0,1) **Grass 2** — same palette, different pattern
- (0,2) **Grass 3** — variant with tiny flower accents
- (0,3) **Grass 4** — variant with a few darker patches

## Row 1 — Dirt:
- (1,0) **Dirt 1** — packed earth, warm brown, small pebbles
- (1,1) **Dirt 2** — different pebble arrangement
- (1,2) **Dirt 3** — lighter, drier variant
- (1,3) **Dirt 4** — variant with grass tufts growing through

## Row 2 — Stone:
- (2,0) **Stone 1** — cobblestone, grey-brown with mossy gaps
- (2,1) **Stone 2** — different stone arrangement
- (2,2) **Stone 3** — darker, more weathered
- (2,3) **Stone 4** — cracked stone with moss in cracks

## Row 3 — Water:
- (3,0) **Water 1** — calm blue surface with ripple highlights
- (3,1) **Water 2** — deeper blue, fewer highlights
- (3,2) **Water 3** — shallow, can see pebbles underneath
- (3,3) **Water 4** — subtle current direction (lighter streak)

## Row 4 — Sand:
- (4,0) **Sand 1** — warm tan beach sand, subtle grain
- (4,1) **Sand 2** — different grain pattern
- (4,2) **Sand 3** — wet sand, slightly darker
- (4,3) **Sand 4** — sand with tiny shell details

## Row 5 — Special terrain:
- (5,0) **Snow 1** — white with blue-grey shadows
- (5,1) **Snow 2** — variant with disturbed surface
- (5,2) **Ice 1** — light cyan, reflective highlights
- (5,3) **Ice 2** — cracked ice, dark crack lines

## Row 6 — More special terrain:
- (6,0) **Swamp 1** — murky green-brown, scattered lily pads
- (6,1) **Swamp 2** — variant, more brown
- (6,2) **Lava 1** — dark red-black crust with bright orange-yellow hotspots
- (6,3) **Lava 2** — more crust visible

## Row 7 — Decorative ground:
- (7,0) **Flower meadow** — grass base covered with red, yellow, purple flowers
- (7,1) **Mossy stone** — stone with heavy green moss overgrowth
- (7,2) **Gravel** — fine grey-brown gravel, uniform small texture
- (7,3) **Dark forest floor** — very dark greens and browns, fallen leaves
