# Base Tiles

4 columns x 8 rows, 32 tile slots. Each cell is 256x128 containing a 2:1 isometric diamond.

## Row 0 — Grass:
- (0,0) **Grass 1** — lush green forest floor, short pixel grass, 3-4 shades of green
- (0,1) **Grass 2** — same palette, different pixel arrangement
- (0,2) **Grass 3** — variant with tiny yellow flower pixels (2-3 flowers)
- (0,3) **Grass 4** — variant with a few darker green patches

## Row 1 — Dirt:
- (1,0) **Dirt 1** — packed earth, warm brown, small pebble pixels
- (1,1) **Dirt 2** — different pebble arrangement
- (1,2) **Dirt 3** — lighter tan variant, drier look
- (1,3) **Dirt 4** — variant with a few grass pixel tufts growing through

## Row 2 — Stone:
- (2,0) **Stone 1** — cobblestone, grey-brown with mossy gaps
- (2,1) **Stone 2** — different stone arrangement
- (2,2) **Stone 3** — darker stone, more weathered
- (2,3) **Stone 4** — cracked stone variant with moss in the cracks

## Row 3 — Water:
- (3,0) **Water 1** — calm blue surface, pixel ripple highlights
- (3,1) **Water 2** — deeper blue, fewer highlights
- (3,2) **Water 3** — shallow water, lighter, can see pebbles underneath
- (3,3) **Water 4** — variant with a subtle current direction (lighter streak)

## Row 4 — Sand:
- (4,0) **Sand 1** — warm tan beach sand, subtle grain
- (4,1) **Sand 2** — different grain pattern
- (4,2) **Sand 3** — wet sand, slightly darker
- (4,3) **Sand 4** — sand with tiny shell pixel details

## Row 5 — Special terrain:
- (5,0) **Snow 1** — white with blue-grey pixel shadows
- (5,1) **Snow 2** — variant with footprint-like disturbed pixels
- (5,2) **Ice 1** — light cyan, reflective pixel highlights
- (5,3) **Ice 2** — cracked ice, dark crack line pixels

## Row 6 — More special terrain:
- (6,0) **Swamp 1** — murky green-brown, scattered lily pad pixels
- (6,1) **Swamp 2** — variant, more brown
- (6,2) **Lava 1** — dark red-black crust with bright orange-yellow pixel hotspots
- (6,3) **Lava 2** — variant, more crust visible

## Row 7 — Decorative ground:
- (7,0) **Flower meadow** — grass base covered with red, yellow, purple flower pixels
- (7,1) **Mossy stone** — stone with heavy green moss overgrowth
- (7,2) **Gravel** — fine grey-brown gravel, uniform small pixel texture
- (7,3) **Dark forest floor** — very dark greens and browns, fallen leaf pixels

## Palette
- Grass: #2d5a1e, #3d7a2e, #5a9a3e, #7aba5e
- Dirt: #5a3a1e, #7a5a2e, #9a7a4e, #baa06e
- Stone: #4a4a4a, #6a6a6a, #8a8a8a, moss #3d5a2e
- Water: #1e3a6a, #2e5a8a, #4e7aaa, #8ebadd
- Sand: #c8a870, #b09060, #e0c890
- Snow: #d8e8f0, #b0c8d8, #ffffff
- Lava: #8a1a0a, #cc4400, #ff8800, #ffcc00
- Shared highlight: #e8d8a8
