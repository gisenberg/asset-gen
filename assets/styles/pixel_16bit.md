# Style: 16-Bit Pixel Art

## Aesthetic
SNES/Sega Genesis era pixel art. Clean, deliberate pixel placement. Every pixel matters.

## Rules
- NO anti-aliasing, NO sub-pixel blending, NO gradients, NO soft edges
- Integer pixels only — no fractional positioning, no blurring
- Limited palette: 8-16 colors per tile for authentic 16-bit feel
- Consistent highlight and shadow colors across all tiles in a set
- Dithering allowed sparingly for texture (checkerboard patterns, not noise)

## Palette Construction
- Each terrain type uses 3-5 color ramps (light → dark)
- All ramps share a common highlight color and a common shadow color
- Adjacent terrain types share at least one midtone for smooth transitions
